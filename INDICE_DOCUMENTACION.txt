================================================================================
📍 FLUJO RECOMENDADO PARA NUEVOS DESARROLLADORES
================================================================================

Paso 1: LEER DOCUMENTACIÓN
   └─> CONFIGURAR_MYSQL.txt (10 minutos)

Paso 2: INSTALAR DEPENDENCIAS
   ├─> Instalar Python 3.8+
   ├─> Instalar XAMPP
   ├─> Crear entorno virtual (.venv)
   └─> pip install -r requirements.txt

Paso 3: CONFIGURAR BASE DE DATOS
   ├─> Iniciar XAMPP MySQL
   ├─> Crear base de datos: frutos_oro_db
   ├─> Importar: database/schema.sql
   ├─> Copiar: .env.example → .env
   └─> Editar .env con tus credenciales

Paso 4: VERIFICAR
   └─> python verificar_conexion.py

Paso 5: EJECUTAR
   └─> python app.py

Paso 6: NAVEGAR
   └─> http://localhost:5000
       Usuario: admin
       Contraseña: admin123

🔐 ARCHIVOS QUE NO DEBES SUBIR A GIT
================================================================================

❌ .env                    ← Tus credenciales (está en .gitignore)
❌ __pycache__/            ← Cache de Python
❌ .venv/                  ← Entorno virtual
❌ uploads/                ← Archivos subidos
❌ *.log                   ← Logs del sistema

✅ .env.example            ← SÍ se sube (sin credenciales reales)
✅ Todos los .py           ← Código fuente
✅ database/schema.sql     ← Script de BD
✅ Documentación (.md, .txt)
